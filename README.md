WHAT COLOR MODEL IS USED ,MENTION THE VALUES
(R,G,B,H,S,L)

RGB Color model
Green= rgb (0,255,0)
R’ = R/255 = 255/255 = 0
G’ = G/255= 0/255= 1
B’ = B/255= 0/255 = 0
Cmax= 1
Cmin = 0
DEL = Cmax -Cmin = 1 – 0 = 1

L= ½ * 1  =  0.50 = 50 %
H= 60 * [2-(R – B) /(G -B)] = 60 * 2 = 120
S = Max(RGB) – Min( RGB) / 1 -| 2L -1|) = 1 – 0 / 1 – ( 2 * 0.50 - 1) = 100 %




Blue= rgb (0 ,0, 255)
R’ = R/255 = 255/255 = 0
G’ = G/255= 0/255= 0
B’ = B/255= 0/255 = 1
Cmax= 1
Cmin = 0
DEL = Cmax -Cmin = 1 – 0 = 1

L= ½ * 1  =  0.50 = 50 %
H= 60 * [4+(R – G) /(B-G)] = 240
S = Max(RGB) – Min( RGB) / 1 -| 2L -1|) = 1 – 0 / 1 – ( 2 * 0.50 - 1) = 100 %
